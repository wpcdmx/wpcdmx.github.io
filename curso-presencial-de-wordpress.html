Curso presencial de WordPress

Crea y gestiona tu propio sitio web, sin necesidad de contratar un programador.

Este cursoEs para ti si:

Quieres aprender a crear tu propia web, para tus proyectos o bien para tus propios clientes.

Si eres Emprendedor, Freelance, Diseñador, Marketer o Bloguero, este curso es un buen punto de partida.

Lo que aprenderás

Se trata de un curso 100% práctico donde aprenderás los conceptos básicos y necesarios para crear y gestionar webs con WordPress. Pero siempre desde la práctica. Probándolo todo en directo.

Preguntas & Respuestas

Este curso de WordPress - WP - esta diseñado para que seas capaz de crear tu propia web, te daré todas las herramientas necesarias para que crees y gestiones tu sitio web.

Si, trabajaras con en tu proyecto web, ya que iremos paso a paso hasta que tu web este terminada.

Si, cuando termines el curso podrás gestionar cualquier tipo de sitio web en WordPress como sitios Corporativos, Marca Personal, Ecommerce, Blogs, etc.

No, no es necesario que tengas conocimientos de WordPress, Diseño o Programación, te daremos los conocmientos y herramientas para que puedas desarrollar tu web.

Si, cuentas con 3 tutorias individuales de 40 minutos y las puedes usar cuando las necesites en un periodo máximo de 3 meses.
